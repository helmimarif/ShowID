# ShowID

### What is this?

* This is a resource that allows you to show your id and nearby player id.

### Features
- Easy configuration.

## Download & Installation

### Using Git
```
cd resources
git clone https://github.com/helmimarif/ShowID
```

### Manually
- Download from this.

## Installation
- Add this to your `server.cfg`:

```
start ShowID
```
